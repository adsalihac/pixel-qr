# PixelQR

PixelQR is an Expo 57 single-page QR code generator.

## Setup

Use pnpm for this project:

```sh
pnpm install
pnpm start
```

Avoid running `npm install` in this workspace. The dependency tree is locked with `pnpm-lock.yaml`, and mixing npm with pnpm can create noisy peer-resolution warnings and partial `node_modules` state.
